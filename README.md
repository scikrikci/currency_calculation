# Currency Calculator

## In this project, you can convert the exchange rate you want to the exchange rate you want.

---
### make

- reate requirements
```bash
make build
```
---
- Start project
```bash
make run
```
---
- Projects API 
```bash
    https://exchangeratesapi.io/
```
---
### Arguments required for the project
```bash
    -f USD & --from_ USD
    -t TRY & --to_ TRY
    -a 40 & --amount 40
```
---
- Test
```bash
    python3 config.py -f EUR -t TRY -a 123
```
